# Nemanja Filipović Portfolio

Fast Astro landing page for Nemanja Filipović, focused on technical consulting services:
business websites, Python automation, data processing, scraping, reconciliation tools,
log analysis utilities, and production support credibility.

## Stack

- Astro
- TypeScript
- Tailwind CSS via `@tailwindcss/vite`
- Static output

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Content

Editable page content lives in `src/data/`:

- `profile.ts`
- `services.ts`
- `projects.ts`
- `experience.ts`
- `skills.ts`

Page sections and reusable presentation components live in `src/components/`.
